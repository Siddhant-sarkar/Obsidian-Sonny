# Wichtig

## im [[nominative]]

| auf Deutsche | auf Englisch          |
| ------------ | --------------------- |
| Ich          | I                     |
| Du           | you __informal__      |
| Er/Sie/Es    | he, she, it           |
| wir          | We                    |
| Ihr          | you __second person__ |
| Sie          | They                  |

## im [[Akkusativ]]
| Nominative | Akkusative | auf englisch         |     |
| ---------- | ---------- | -------------------- | --- |
| ich        | mich       | me                   |     |
| du         | dich       | you                  |     |
| er         | ihn        | him                  |     |
| sie        | sie        | her                  |     |
| es         | es         | it                   |     |
| wir        | uns        | us                   |     |
| ihr        | euch       | you(group of people) |     |
| sie        | sie        | them                 |     |
